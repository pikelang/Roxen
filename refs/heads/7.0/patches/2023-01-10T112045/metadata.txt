• Git: Begin migration to using only spaces for indentation.

This commit sets the git attribute "space_indent" on all files,
defines it to use the git filter "space-indent" and adds a list
of exceptions (that also get the attribute "mixed_indent").

The space-indent filter is typically defined with:
$ git config --global filter.space-indent.clean 'unexpand --first-only'
$ git config --global filter.space-indent.smudge 'cat'

Prepares for [REP-817].
